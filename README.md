# Análisis Exploratorio de Datos
### Fernando Gutiérez Canales

Este repositorio contiene el código de una aplicación web dedicada al análisis de datos exploratorios de un dataset sobre ventas de automóviles. ¡Con esta aplicación el usuario será capaz de hacer sus primero análisis sobre los datos de venta de automóviles en Estados Unidos!

El repositorio forma parte del bootcamp de Data Analyst de Tripleten.
